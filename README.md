# agentsystems-sdk

```bash
python3 -m pip install --upgrade pipx  # once
pipx install agentsystems-sdk          # creates ~/.local/pipx/venvs/agentsystems-sdk
```

```bash
agentsystems --version
```

```bash
# remove the PyPI build if you have it
pipx uninstall agentsystems-sdk

# install from source (editable by default)
pipx install --editable .  # run this in the repo root
agentsystems --version
```
